Preprocesadores

¿Para qué sirven?

Los preprocesadores son herramientas que transforman diversos lenguajes de código en HTML, CSS o JavaScript, de forma que el navegador pueda comprenderlos. También permiten adaptar versiones modernas de un lenguaje para garantizar su adaptación a cualquier navegador.

Los preprocesadores se basan en directrices, entre las cuales encontramos:

  • Macro expansiones (#define)
  • Inclusión de archivos (#include)
  • Compilaciones condicionales (#if, #ifdef, #endif...)
  • Mensajes de error (#error)

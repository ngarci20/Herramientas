PostCSS

PostCSS es una herramienta para la transformación de código CSS mediante plugins basados en JavaScript. Así, PostCSS permite implementar una serie de plugins, como Autoprefixer (para la compilación de CSS moderno), o RTLCSS (para cambiar el sentido de lectura de los archivos CSS a right-to-left).

Los plugins son, en esencia, los preprocesadores que optimizan el código, añaden sintaxis, realizan el linting (detección de errores), procesan las variables, etc.

¿Qué hace especial a PostCSS?

  • Diversidad de funcionalidades de los plugins
  • Permite usar solo los plugins que necesitemos
  • Rapidex de ejecución
  • Permite la creación de plugins propios
  • Sus librerías no están vinculadas a un preprocesador

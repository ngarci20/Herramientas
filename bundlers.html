Module Bundlers

¿Para qué sirven?

Ejecutar Bundle

Los module bundlers son herramientas front-end que permiten a los desarrolladores agrupar archivos de scripts y estilos para una mayor optimización de sus páginas. Además, en un entorno de desarrollo, permite lanzar un servidor de desarrollo y monitorizar los cambios en los archivos para refrescar el navegador automáticamente.

¿Por qué es necesario implementar un Module Bundler?

En primer lugar, ayuda a gestionar las relaciones de dependencia de nuestro código, ya que los módulos se cargarán siguiendo este orden de dependencia. También permitirá optimizar imágenes, código CSS y demás assets.

Las tareas que automatiza un module bundler pueden variar en función del entorno de destino de nuestra aplicación. Actualmente, los module bundlers realizan estas tareas de forma transparente al usuario, aunque permiten la configuración para casos especiales.
